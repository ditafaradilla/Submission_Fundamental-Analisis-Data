E-Commerce Performance Dashboard - Analysis Project
Analist: Dita Faradilla
Tahun: 2026

Dashboard ini dirancang untuk menganalisis performa penjualan e-commerce 
menggunakan dataset publik, mencakup analisis kategori produk unggulan 
dan segmentasi pelanggan menggunakan metode RFM.

-----------------------------------------------------------
1. PRASYARAT (PREREQUISITES)
-----------------------------------------------------------
Pastikan Anda sudah menginstal Python (versi 3.9 atau lebih baru).
Library utama yang digunakan:
- Streamlit
- Pandas
- Matplotlib
- Seaborn

-----------------------------------------------------------
2. INSTALASI LIBRARY
-----------------------------------------------------------
Setelah environment aktif, instal semua dependensi yang diperlukan:

pip install -r requirements.txt

-----------------------------------------------------------
3. CARA MENJALANKAN DASHBOARD
-----------------------------------------------------------
Pastikan file 'main_data.csv' berada dalam satu folder yang sama 
dengan file dashboard ('dashboard.py' atau nama file utama Anda).

Jalankan perintah berikut di terminal:

streamlit run nama_file_anda.py

-----------------------------------------------------------
4. STRUKTUR FILE
-----------------------------------------------------------
- nama_file_anda.py : File utama kode Streamlit.
- main_data.csv     : Dataset yang telah dibersihkan.
- requirements.txt  : Daftar library yang dibutuhkan.
- README.txt        : Instruksi penggunaan.
